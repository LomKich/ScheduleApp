ScheduleApp — Папка звуков
===========================
Все звуки приложения хранятся в этой папке.
Вы можете заменить любой файл своим.
Поддерживаемые форматы: .ogg (рекомендуется), .mp3

СПИСОК ЗВУКОВ:
--------------
msg_send.ogg        — Отправка сообщения
msg_receive.ogg     — Получение сообщения
btn_click.ogg       — Нажатие кнопки / общий звук UI
nav_screen.ogg      — Переход между экранами
theme_select.ogg    — Смена темы / выбор настройки
success.ogg         — Успех / выполнено
error.ogg           — Ошибка / запрет
toast.ogg           — Toast-уведомление
game_start.ogg      — Начало игры / старт
game_over.ogg       — Конец игры / поражение
snake_eat.ogg       — Змейка: съела еду
tetris_line.ogg     — Тетрис: убрана линия
pong_hit.ogg        — Пинг-понг: удар по шарику
bubble_pop.ogg      — Пузыри: лопнул пузырь
screamer.ogg        — Скример: страшный звук (пасхалка в играх)

КАК ЗАМЕНИТЬ ЗВУК:
1. Создайте или найдите звуковой файл в формате .ogg
2. Переименуйте его точно так как указано выше
3. Скопируйте в эту папку (заменив старый файл)
4. Пересоберите APK

───────────────────────────────────────────────────────────
КАСТОМНЫЙ СКРИМЕР (изображение)
───────────────────────────────────────────────────────────
Вы можете заменить стандартное SVG-лицо скримера
на собственное изображение!

ФАЙЛ: screamer.jpg  (или screamer.png)
Положите ваш файл в эту папку. Поддерживаются:
  • screamer.jpg — JPEG изображение
  • screamer.png — PNG изображение (с прозрачностью)
  • screamer.gif — GIF (анимированный)
  • screamer.webp — WebP

Рекомендуемый размер: 400×400 пикселей или больше.
Приложение автоматически определит наличие файла и
использует его вместо встроенного SVG.

Если файл отсутствует — используется встроенное SVG-лицо.
