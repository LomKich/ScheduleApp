# CORS-прокси для ScheduleApp

## Зачем нужен прокси?

Android WebView блокирует прямые запросы к API Яндекс Диска из-за CORS.
Прокси-сервер выступает посредником: приложение → прокси → Яндекс API.

## Деплой на Cloudflare Workers (бесплатно)

1. Зарегистрируйся: https://dash.cloudflare.com (бесплатный план — 100 000 запросов/день)
2. Перейди: **Workers & Pages** → **Create** → **Create Worker**
3. Скопируй содержимое `worker.js` → вставь в редактор → **Deploy**
4. Скопируй URL воркера (например: `https://schedule-proxy.username.workers.dev`)
5. Открой приложение → **Настройки** → **Прокси-сервер**
6. Вставь: `https://schedule-proxy.username.workers.dev/proxy/`
7. Нажми **Сохранить прокси**
8. На главной нажми кнопку **Обновить** или вернись на главную

## Проверка

После настройки в верхней части настроек должно появиться:
✅ Прокси: https://your-proxy.workers.dev/proxy/
